# 🚀 CareerVerse AI
https://careerverse-ai-syuavvllgkab242ibt4mio.streamlit.app/

## Your Personal Career Simulation Platform

CareerVerse AI is a creative career-planning and future-visualization platform designed to help students explore their professional journey through interactive simulations, career roadmaps, mentorship guidance, interview preparation, and skill tracking.

Built for the GitHub Copilot Creative Apps Hackathon.

---

## 🌟 Problem Statement

Many students struggle to:

- Visualize their future career path
- Identify skills needed for their dream job
- Prepare for interviews
- Build confidence in their career journey
- Stay motivated throughout their learning process

CareerVerse AI transforms career planning into an engaging and interactive experience.

---

## 🎯 Solution

CareerVerse AI helps students:

✅ Analyze resumes

✅ Generate future career simulations

✅ Create personalized career roadmaps

✅ Learn from virtual mentors

✅ Practice interview questions

✅ Track skill development

✅ Calculate career readiness

---

## ✨ Features

### 📄 Resume Analyzer

- Upload PDF resume
- Detect technical skills
- Generate ATS-style score
- Identify existing strengths

---

### 🔮 Future Self Generator

Visualize your future career journey.

Example:

> You are a successful AI Engineer in 2030 working on cutting-edge technologies and mentoring aspiring developers.

---

### 🛣 Career Story Generator

Creates a step-by-step career roadmap.

Example:

- Learn Programming
- Build Projects
- Complete Internships
- Contribute to Open Source
- Become an Industry Professional

---

### 🧑‍🏫 AI Mentor

Choose your mentor:

- Google Recruiter
- AI Engineer
- Startup Founder

Receive personalized guidance and career advice.

---

### 🎤 Interview Simulator

Practice interview questions in:

- Python
- Artificial Intelligence
- General HR Interviews

---

### 📊 Skill Dashboard

Visualize:

- Python Skills
- AI Skills
- Git Skills
- Communication Skills

Includes achievement badges and progress tracking.

---

### 🎯 Career Readiness Calculator

Calculate your career readiness score based on:

- Technical Skills
- Communication
- Projects

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/yourusername/CareerVerse-AI.git
```

Move into project folder:

```bash
cd CareerVerse-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
